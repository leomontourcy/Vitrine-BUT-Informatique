# Développement Web

Site web statique multi-pages réalisé en équipe de 4 étudiants dans le cadre d’un projet de BUT Informatique.
Lien : https://leomontourcy.github.io/Vitrine-BUT-Informatique/

## 📖 À propos

Ce projet consiste en la réalisation d’un site web statique en HTML5/CSS3, hébergé sur GitHub Pages.  
Il présente la formation BUT Informatique, ses compétences, sa présentation générale et une page de contact.

L’objectif est de mettre en pratique les compétences acquises en développement web front-end, tout en respectant une structure propre, responsive et facile à naviguer.

## ✨ Fonctionnalités

- Site multi-pages
- Navigation entre les différentes sections
- Design responsive
- Pages dédiées à l’accueil, la présentation, les compétences et le contact
- Structure simple et claire
- Hébergement sur GitHub Pages

## 🛠️ Technologies

- HTML5
- CSS3
- Git / GitHub
